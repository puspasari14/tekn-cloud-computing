### Cloud computing

cloud computing adalah menyediakan sumber daya sistem komputer sesuai dengan perimntaan terutama penyimpanan data dan daya komputasi oleh pengguna. Pusat data tersedia bagi banyak pengguna melalui internet. memilki fungsi yang didistribusikan di beberapa lokasi server pusat.

Pendukung cloud publik dan hybrid mencatat bahwa cloud computing memungkinkan perusahaan untuk menghindari atau meminimalkan biaya infrastruktur TI di muka.Ketersediaan jaringan berkapasitas tinggi, komputer murah dan perangkat penyimpanan serta adopsi luas virtualisasi perangkat keras, arsitektur berorientasi layanan, dan komputasi otonom dan utilitas telah menyebabkan pertumbuhan dalam komputasi awan.
